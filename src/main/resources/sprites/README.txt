EscoltaCore — sprites/
======================
Place PNG files named after Bukkit Material names (UPPERCASE) here.
Examples:  DIAMOND.png   GOLD_INGOT.png   OAK_LOG.png

On first startup, missing sprites are auto-downloaded from:
  https://github.com/Faithful-Pack/Default-Java  (1.21.5 branch)

If the server has no internet access, a colored fallback swatch
is generated automatically so the map item always shows something.

Downloaded files are cached here for future restarts.
